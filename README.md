# 線対称・点対称ビジュアライザー

小学校の算数で、先生が電子黒板やプロジェクターに映して使うことを想定したWebアプリです。
線対称と点対称を切り替えながら、1つの図形に対称の軸や対称の中心を重ねて考えられます。

## 使い方

GitHub PagesではURLを開くだけで使えます。

手元で確認するときに `index.html` を直接開けない場合は、次のコマンドを使います。

```bash
npm start
```

その後、ブラウザで `http://127.0.0.1:8003/` を開きます。

- 「線対称」「点対称」タブで表示を切り替えます。
- 線対称モードでは、赤い線の端をドラッグして対称の軸を予想します。
- 点対称モードでは、赤い中心をドラッグして対称の中心を予想します。
- 「確認する」で予想を簡単に判定できます。
- 「答えを表示」で、正しい対称の軸や中心を重ねて見せられます。
- 対応の手がかり、点のラベル、方眼、答えの強調表示をオン/オフできます。
- 「使い方」から、授業での使い方と簡単な授業例を確認できます。

## 図形の例

- たこ形: 線対称だけ
- ひし形: 線対称、点対称
- 長方形: 線対称、点対称
- 正方形: 線対称、点対称
- 平行四辺形: 点対称だけ
- 二等辺三角形: 線対称だけ

## GitHub Pagesで無料公開する流れ

1. GitHubで公開リポジトリを作成します。
2. このフォルダー内のファイルをアップロードします。
3. リポジトリの Settings から Pages を開きます。
4. Source を main ブランチの root に設定します。
5. 表示された `https://ユーザー名.github.io/リポジトリ名/` のURLを先生に渡します。

このアプリはサーバーや外部APIを使わない静的Webアプリなので、GitHub Pagesにそのまま載せられます。
